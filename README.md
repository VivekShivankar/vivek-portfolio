# Vicky's Portfolio - Vivek Sunil Shivankar

A premium, professional static portfolio website built with HTML, CSS & JavaScript.

## How to Run in Eclipse

1. **Extract the zip file** to your computer
2. Open **Eclipse IDE**
3. Go to **File → New → Static Web Project** (or any web project)
4. Copy the extracted files into your project folder
5. Right-click `index.html` → **Open With → Web Browser**
   OR simply open `index.html` directly in any browser (Chrome, Firefox, Edge)

## Contact Form

The contact form uses **FormSubmit.co** — a free service that forwards messages directly to:
`shivankarvivek44@gmail.com`

**First-time setup:** When someone submits the form for the first time, FormSubmit may send a confirmation email to activate. Just click "Confirm" in that email and all future messages will arrive directly.

No server or backend required — works 100% static!
